# DEVCONF 2026

A modern and responsive conference landing page built using **HTML5** and **CSS3**. This project is designed for the DevConf 2026 assignment and follows the provided Figma design.

## Live Preview

https://alamindev07.github.io/DevConf-2026/

## Features

- Responsive Navigation Bar
- Hero Banner Section
- Meet the Speakers Section
- Pricing Section
- Sponsors & Partners Section
- Footer with Social Media Links
- Fully Responsive Design
- Clean and Semantic HTML5
- Pure CSS3 (No Framework)

## Technologies Used

- HTML5
- CSS3
- Font Awesome (Icons)
- Google Fonts (Inter)

No installation or additional setup is required.
